# Páginas del sitio

## index.html
Página principal del proyecto. Presenta el objetivo del sitio y permite acceder al resto de páginas.

## sostenibilidad.html
Explica qué es la sostenibilidad y por qué es importante para el desarrollo actual y futuro.

## economia-circular.html
Describe el concepto de economía circular y cómo se diferencia del modelo económico tradicional.

## ciclo-vida.html
Muestra las diferentes fases del ciclo de vida de un producto mediante un diagrama interactivo.

## mapa.html
Simulación de puntos de reciclaje donde el usuario puede interactuar y ganar puntos.

## producto.html
Presenta un ejemplo de producto sostenible diseñado bajo principios de economía circular.

## equipo.html
Página con información sobre el equipo o los participantes del proyecto.

# Tecnologías utilizadas

El proyecto está desarrollado con tecnologías web básicas:

- **HTML5** → estructura de las páginas  
- **CSS3** → diseño y estilos visuales  
- **JavaScript** → interactividad de la página  

También se utiliza:

- **Normalize.css** para mejorar la compatibilidad entre navegadores.

---

# Funcionalidades

El proyecto incluye algunas funcionalidades interactivas:

- **Acordeones de información** en algunas secciones.
- **Diagrama interactivo del ciclo de vida** que muestra información al pasar el ratón.
- **Sistema de puntos en el mapa de reciclaje** al interactuar con los puntos.

Estas funcionalidades están implementadas en el archivo:
js/main.js
